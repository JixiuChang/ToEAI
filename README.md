# ToEAI Frontend

This repository contains a VueÂ 3 + Vite frontend for the Tales of Echo AI chatbot. This is the pure frontend.

## Development

Install dependencies and start the dev server:

```bash
pnpm install
pnpm dev
```

Build for production:

```bash
pnpm build
pnpm preview
```

The app will be served on <http://localhost:5173> by default.
